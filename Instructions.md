## Brand Voice Guide Testing Protoco

### Immediate Testing Protocol

1. **Generate Test Content (3 pieces):**
   ```
   Using the Brand Voice Guide you just created, generate:
   
   1. A 500-word educational blog post on [TOPIC FROM THEIR ALTC]
   2. A 200-word service page introduction for [THEIR PRIMARY SERVICE]
   3. A 100-word social media post about [RECENT PROJECT OR INSIGHT]
   
   Apply all humanisation techniques from the guide.
   ```

2. **Run AI Detection:**
   - Test all 3 pieces through your preferred AI detector
   - Target: Each piece should score 6-15%
   - If any piece scores >15%, note which sections triggered it

3. **Refinement Loop (if needed):**
   ```
   The [CONTENT TYPE] scored [X]% on AI detection, with flags in:
   - [Section/element that triggered detection]
   
   Please refine the voice guide's guidance on [THAT ELEMENT] to reduce detection further.
   Then regenerate the test content with the updated guidance.
   ```

4. **Client Review:**
   - Share the 3 test pieces with client (not the voice guide)
   - Ask: "Does this sound like you? What would you say differently?"
   - Use their feedback to refine specific sections

5. **Finalize & Deploy:**
   - Lock in the refined voice guide
   - Add it to client's Claude Project Knowledge
   - Use for all ALTC content generation going forward
