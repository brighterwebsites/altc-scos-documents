# **Brand Alignment Checklist (Website ↔ Social)**

**Purpose:** To evaluate how consistently a business presents its brand across **Instagram, Facebook, and their website**. This alignment directly impacts trust, conversion, and professionalism.

Use this framework when reviewing screenshots from both platforms.

---

# **1\. Scoring Overview (0–10)**

Use this scoring matrix for the final Brand Alignment Score.

| Score | Meaning | Interpretation |
| :---- | :---- | :---- |
| **0–3** | Poor Alignment | Major inconsistencies. Visual identity unclear. Impacts trust. |
| **4–6** | Partial Alignment | Some consistency, but mixed quality. Needs refinement. |
| **7–8** | Strong Alignment | Looks cohesive across platforms; small improvements needed. |
| **9–10** | Excellent Alignment | Highly professional, intentional, consistent everywhere. |

**Rule:** If parts of the brand system are not visible due to limited screenshots, write:

“Cannot determine from screenshots provided.”

---

# **2\. Assessment Areas**

## **A. Visual Alignment**

Evaluate:

* Colours  
* Fonts  
* Logo usage  
* Photography style  
* Editing/graphics style  
* Overall cohesiveness

### **What Good Looks Like**

**Strong alignment** examples:

* Website uses deep blue \+ gold; social posts use the same palette  
* Same fonts or close stylistic match  
* Photography style consistent (lighting, tone, composition)  
* Logo appears clean and similar across all platforms  
* Feed has a deliberate aesthetic (not chaotic)

### **Weak alignment examples:**

* Website modern; socials look DIY  
* Website uses clean professional images; socials use blurry photos  
* Logo appears differently sized, stretched, or inconsistent  
* Colours clash between web and socials  
* Website aesthetic is minimal; socials are chaotic or cartoonish

### **How to Assess from Screenshots**

* Compare the HEX-like colour families (not exact values)  
* Look for consistent type styles (serif/sans-serif/script)  
* Identify repeated visual motifs (rounded edges, boxes, gradients, etc.)  
* Check for consistency in image tone, exposure, contrast  
* Look for repeated brand devices (icons, linework, overlays)

---

## **B. Messaging Alignment**

Evaluate:

* Value proposition  
* Tone of voice  
* Core services  
* “Why choose us” messaging  
* Offer structure

### **What Good Looks Like**

* Website headline: “Premium Landscaping Ballarat.” Social captions reference: “Transforming Ballarat gardens…”  
* Website tone calm and expert; socials match  
* Website emphasises 3 key services; socials show those same services  
* Offers match (pricing, inclusions)

### **Weak alignment examples**

* Website says “luxury” but socials feel budget  
* Website tone professional; socials meme-heavy  
* Website lists 5 services; socials only show 1 or show unrelated content  
* Offers differ (“10% off” on socials but not on site)

### **Screenshot Assessment Tips**

* Compare homepage value prop vs IG bio  
* Compare website H1/H2 wording with Facebook “About”  
* Scan captions for alignment with core brand language  
* Check highlights for CTA or service references

---

## **C. Trust Alignment**

Evaluate:

* Social proof consistency  
* Quality standards  
* Professionalism  
* Before/after consistency  
* Alignment of “promises vs. proof”

### **Strong trust alignment examples**

* Reviews featured on website and reposted on socials  
* Before/after showcased across both platforms  
* Visual professionalism matches (no downgrade on socials)  
* Tone of trust: calm, expert, confident

### **Weak trust alignment examples**

* Website looks premium; socials look low-effort  
* Website shows polished finished work; socials show inconsistent quality  
* Website has reviews; socials have none  
* Promises on website not supported by social content

### **Screenshot Assessment Tips**

* Compare the standard of work shown on socials to the claims on the website  
* Look for review screenshots or social proof content  
* Check storytelling quality (case studies, outcomes, results)

---

# **3\. Interpretation Framework**

After reviewing Visual \+ Messaging \+ Trust:

### **If most elements align → Score 7–10**

Brand feels cohesive, professional, and intentional.

### **If there are pockets of inconsistency → Score 4–6**

Brand is on the right track but lacks polish or consistency.

### **If visual \+ messaging clash → Score 0–3**

Misalignment is harming credibility and conversions.

---

# **4\. Recommended Fixes (Based on Common Scenarios)**

Use these when generating recommendations.

## **Scenario 1: Website polished, social low-quality**

* Create 10–20 high-quality branded templates  
* Replace low-res images with new branded visuals  
* Introduce a cohesive colour palette  
* Standardise fonts using accessible equivalents  
* Add proof-driven posts (reviews, before/after)

## **Scenario 2: Social strong, website outdated**

* Refresh homepage hero section to match current visual identity  
* Update fonts and colour palette  
* Add social-style photography to the website  
* Ensure CTA style matches socials  
* Add recent reviews/UGC to the site

## **Scenario 3: Messaging mismatch**

* Rewrite IG bio to match the website’s value prop  
* Align service naming conventions  
* Standardise CTA language  
* Ensure highlight titles use consistent messaging  
* Update FB About section to match website tone

## **Scenario 4: Trust mismatch**

* Reshoot key service images  
* Add social proof to socials or website where missing  
* Harmonise brand promises (price, quality, timeframe)  
* Create a “proof library” for cross-platform use

---

# **5\. When to Mark as “Cannot Determine”**

Use this if:

* Font family is not clear  
* Website page content is cut off  
* IG grid only shows a few posts  
* Logo not visible  
* Homepage screenshot does not reflect deeper brand elements  
* Facebook page sections not included

**Never infer or invent missing content.**
